# Days Since Tracker

A Flutter mobile app to track days since important events.

## Project Details
- **Platform**: Android (Google Play Store)
- **Framework**: Flutter
- **Target Launch**: January 25, 2026

## Features (MVP)
- Track multiple events with custom names
- Day counter with color-coded status indicators
- Pre-set templates for common tracking needs
- Progress ring visualization
- Local data persistence (Hive)
- Dark/light theme support

See `mater-plan.md` in the root directory for the complete timeline.

## Getting Started

```bash
# Get dependencies
flutter pub get

# Run the app
flutter run
```
